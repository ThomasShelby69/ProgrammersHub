# ProgrammersHubBot
*A bot for Programmers Hub discord server.*

## Features:
* Core / Information Commands - 7
* Moderation Commands - 3
* Music Commands - 15
* More coming soon!

## Commands:
### Core / Information Commands
* `botinfo`
* `serverinfo`
* `whois`
* `help`
* `prefix`
* `ping`
* `uptime`

### Moderation Commands
* `ban`
* `kick`
* `purge`

### Music Commands
* `back`
* `clear`
* `filter`
* `loop`
* `nowplaying`
* `pause`
* `play`
* `progress`
* `queue`
* `resume`
* `save`
* `search`
* `skip`
* `stop`
* `volume`
